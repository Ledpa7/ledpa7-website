# Led.발광다이오드 웹사이트 프로젝트 작업 요약 (2026-01-30)

## 1. 프로젝트 개요
산업용 알루미늄 프로파일을 활용한 디자인 브랜드 `Led.발광다이오드`의 공식 웹사이트 구축 프로젝트입니다. 미니멀한 디자인과 3D 가구 모델링 인터랙션을 중심으로 제작되었습니다.

## 2. 주요 구현 기능 및 디자인
### 🎨 인터페이스 (UI/UX)
- **Hero 섹션**: 흑백 대비의 강렬한 레이아웃과 `100 - 100 = ∞` 슬로건을 활용한 임팩트 있는 첫 화면.
- **헤더 시스템**: 스크롤에 최적화된 마이크로 헤더(h-12)와 미니멀한 로고/장바구니 아이콘 배치.
- **네비게이션**: 좌측 고정형 수직 네비게이션으로 섹션 간 이동 편의성 제공.
- **커스텀 커서**: 브랜드 아이덴티티를 강조하기 위한 전체 마우스 커서 숨김 및 커스텀 원형 커서 적용.

### 🔌 기술적 특징
- **3D 인터랙션**: `React Three Fiber` 및 `Three.js`를 사용한 알루미늄 프로파일 가구 3D 모델(001~003) 구현. 마우스 드래그로 회전 및 확인 가능.
- **반응형 그리드**: 데스크탑 3분할 뷰와 모바일 최적화 스택 구조를 동시 구현.
- **스택**: Vite + React + TypeScript + Tailwind CSS 기반의 고성능 아키텍처.

### 📑 정보 구조 및 연동
- **비즈니스 정보**: `https://www.ledpa7.com/` 원본 사이트의 사업자 정보(상호, 대표, 사업자 번호 등) 및 푸터 레이아웃 완벽 이식.
- **외부 링크 연동**: 네이버 스마트스토어(`https://smartstore.naver.com/led-`) 공식 쇼핑몰 연결.

## 3. 향후 유지보수 참고사항
- **3D 모델 교체**: 추가 모델링 업로드 시 `.glb` 또는 `.gltf` 확장자 사용을 권장함. (`src/components/FurnitureCanvas.tsx` 수정)
- **실행 환경**: 프로젝트 루트에서 `npm run dev` 실행.
