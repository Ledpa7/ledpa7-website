# Led.발광다이오드 Official Website

산업에서 쓰이는 알루미늄 프로파일을 활용하여 미니멀한 가구와 디자인 오브제를 제작하는 브랜드 **Led.발광다이오드**의 공식 웹사이트입니다.

## 🚀 프로젝트 개요
- **브랜드명**: Led.발광다이오드
- **슬로건**: `100 - 100 = ∞` (모든 디자인은 100개 한정 판매 후 오픈소스로 전환)
- **주요 기술**: React 19, TypeScript, Vite, Framer Motion, Tailwind CSS, React Three Fiber

## ✨ 핵심 기능
- **Immersive Video Scrubber**: 스크롤에 반응하여 재생되는 고화질 시퀀스 비디오.
- **Interactive Grid**: 마우스 호버 시 활성화되는 제품 영상 인터랙션.
- **Minimalistic UI**: 흑백 대비와 정밀한 그리드 시스템을 활용한 브랜드 아이덴티티 구축.
- **Micro-interactions**: 부드러운 스크롤 애니메이션과 커스텀 커서 시스템.

## 🛠 배포 방법 (Recommended: Vercel)
현 프로젝트는 SPA(Single Page Application) 구조로 제작되었습니다. Vercel 배포 시 `vercel.json` 설정이 이미 포함되어 있어 즉시 배포가 가능합니다.

1. `dist` 폴더 빌드: `npm run build`
2. 빌드된 `dist` 폴더를 Vercel 또는 Netlify에 업로드합니다.
